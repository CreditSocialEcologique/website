Se connecter à la page utilisateur

Chaque utilisateur de l'application possède sa propre page sur laquelle est récapitulé :

  • son CSE et son score dans chacune des catégories
  • l'historique de ses notifications
  • son profil détaillé

Un utilisateur peut se connecter à sa page dédiée via l'adresse : http://intensif05.ecole.ensicaen.fr:8180/client/login.

Pour se connecter : entrez le nom et le prénom (sensible à la casse) d'un utilisateur et vous serez connecté à sa page personnelle.

Si vous souhaitez vous connecter mais que vous ne connaissez pas le nom d'un des utilisateurs de la base de données, vous pouvez vous connecter au profil de l'utilisateur par défaut "Roche Etienne". Sinon, vous pouvez récupérer le .json représentant les données de la base de données au endpoint : http://intensif05.ecole.ensicaen.fr:8080/api/users.

Se connecter à la page admin

Les admins de l'application ont accès à tous les profils des utilisateurs et à des informations personnelles sur ceux-ci. Pour se connecter au panel admin, se rendre à l'adresse suivante : http://intensif05.ecole.ensicaen.fr:8180/

Points d'entrée de l'application

Détecteur de son

L'arduino envoie 0 ou 1 toutes les secondes. Cela permet de savoir si on a dépassé le seuil de nuisance sonore (reglé avec le potentiomètre).

L'application va ensuite lire ce qui a été envoyé et selon ce qui a été reçu, va faire une requête pour diminuer le score sonore de l'utilisateur. Pour la démonstration, l'utilisateur qui est pénalisé est par défaut l'utilisateur n°1 : "Roche Etienne".

Détecteur de luminosité

L'arduino envoie une valeur comprise entre 0 et 1023 toutes les secondes. Plus une valeur est proche de 0, plus le bruit détecté est élevé. À l'inverse, une valeur proche de 1023 correspond à du silence.

L'application va récupérer les valeurs envoyées et si la valeur est inférieure à 100 (valeur choisie pour la démo), va faire une requête pour diminuer le score luminosité de l'utilisateur. Pour la démonstration, l'utilisateur qui est pénalisé est par défaut l'utilisateur n°1 : "Roche Etienne".

Autres outils

Caméra

À l'aide de la librairie libre OpenCV, un module de reconnaissance d'objets et de personne a été développé. Le module est capable de détecter divers objets ainsi que les êtres humains. Pour des raisons de temps, ce module n'est pour l'instant pas relié à l'application.
